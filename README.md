# Workflow Automation Engine

A Workflow Automation Engine built using **FastAPI** that allows users to design workflows, define steps and rules, execute processes dynamically, and track execution logs through an interactive web interface.

## Features

- Create workflows
- Add workflow steps
- Define rules for workflow execution
- Execute workflows dynamically
- Visual workflow diagram
- Execution logs

## Tech Stack

Backend: FastAPI (Python)  
Frontend: HTML, CSS, JavaScript  
Deployment: Render  
Version Control: GitHub

## How to Run Locally

1. Install dependencies

pip install -r requirements.txt

2. Start server

uvicorn main:app --reload

3. Open browser

http://127.0.0.1:8000
